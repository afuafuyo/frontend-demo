1. 图片的像素数据是用一个一维数组保存在 ImageData 对象中的

-----
|X X|
|X X|
-----

上面是一个 2 * 2 像素的图片，在 ImageData 中，一个像素由 rgba 四部分组成，每部分由 0-255 的数字表示
所以上面可以表示为

--------------
|(rgba)(rgba)|
|(rgba)(rgba)|
--------------

从上可知 2 * 2 的图片需要一个长度为 16 的一维数组来存储像素数据

一张 2 * 2 的半透明红色图片数据为

[
      第一个像素      第二个像素
    ------|------   ------|------
    255, 0, 0, 128, 255, 0, 0, 128,
    255, 0, 0, 128, 255, 0, 0, 128
    ------|------   ------|------
      第三个像素      第四个像素
]
