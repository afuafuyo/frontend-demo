shell 引号

引号必要性

由于 shell 脚本的特殊性，一些字符串如果包含有空格，shell 在执行时可能会误认为这是命令的一部分而执行造成错误，为了防止 shell 这样做，可以使用引号

双引号

普通用法

name="zhangsan"

双引号里面的特殊字符和引用值会被解释执行。包括

  • $
  • `
  • \
[afu@localhost shells]$ name="zhangsan"
[afu@localhost shells]$ echo "$name"
zhangsan

单引号

单引号与双引号类似，不同的是 shell 会忽略任何引用值

[afu@localhost shells]$ echo '$name'
$name

反引号

反引号用于设置系统命令的输出到变量

反引号中必须是一个系统支持的命令

[afu@localhost shells]$ echo `date`
2018年 02月 10日 星期六 16:34:57 CST

[afu@localhost shells]$ echo `lisi`
-bash: lisi: command not found

反斜线

用于转义特殊字符，屏蔽其特殊含义

比如& * + ^ 1 ` " | ?

# * 在 shell 中有特殊含义 想输出 * 字符必须转义
[afu@localhost shells]$ echo *
data.txt student.txt test.sh

[afu@localhost shells]$ echo \*
*
