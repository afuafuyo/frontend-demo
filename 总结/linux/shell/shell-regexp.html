字符串匹配

当你在使用命令行时，有很多时间都用来查找你所需要的文件

shell 提供了一套完整的字符串模式匹配规则

* 匹配

使用星号 * 可以匹配任何字符串

[afu@localhost shells]$ ls *.txt
rs.txt  time.txt

? 匹配

使用 ? 可以匹配任何单个字符

[afu@localhost shells]$ ls ??.txt
rs.txt

使用 [] 和 [!]

使用 [] 可以用来匹配方括号 [] 中的任何一个字符

[afu@localhost shells]$ ls [0-9]*.log
2018.log

正则表达式

使用句点匹配单字符

句点 '.' 可以匹配任意单字符

[afu@localhost shells]$ ls | grep .em.
demo

匹配开头

^ 只允许在一行的开始匹配字符或单词

[afu@localhost shells]$ ls | grep ^de
demo

匹配结尾

$ 在行尾匹配字符串或字符

[afu@localhost shells]$ ls | grep mo$
demo

模式出现几率

# d 字符是否出现 2 次或以上
[afu@localhost shells]$ echo "abcdddef" | grep 'd\{2,\}'
abcdddef
